# ATLAS 新機能評価テンプレート

新機能のプロンプト・仕様・実装を始める**前に**、以下を埋めてレビューしてください。

**最重要判断:** 「この機能は、お客様の習慣的な作業を減らせるか？」 — YES なら追加、NO なら見送り。

## 記入例（SNS投稿の7日分まとめ生成）

機能名：SNS 7日分まとめ生成（エコモード）

ユーザー価値：毎日AIを起動せず、1回の生成で1週間分の投稿文案を用意できる

差別化：競合の「毎回生成」型SNS支援と比べ、コストと手間を抑えつつ予約投稿まで一気通貫

AI必要度：中 — 文案の創作・トーン調整には必要。スケジュール計算・保存は不要

AIなしで実装可能：一部 — 日付ラベル付け・予約保存・ON/OFFは通常プログラム

運営コスト：週1回のLLM呼び出し（日次7回 → 1回に集約、約85%削減見込）

外部APIコスト：無（投稿API連携時は別途 — 連携時のみ従量）

コスト削減案：エコモード + 7日まとめ + キャッシュ + 予約投稿 + 承認後実行オプション

優先度：P1

## 判断基準（クイックリファレンス）

| 質問 | AI不要の例 | AIが必要な例 |
|------|-----------|-------------|
| 定期実行 | cron / nextRun 計算 | — |
| 履歴・状態管理 | DB / localStorage | — |
| 通知・リマインド | 条件分岐 + テンプレート | 文面の都度生成 |
| 曜日・日付判定 | `Date` / timezone ロジック | — |
| ON/OFF・提案条件 | ルールエンジン / if 文 | 曖昧な自然言語判断 |
| コンテンツ生成 | テンプレート差し込み | 記事・提案・要約・判断 |
